# roger--han.github.io

Personal CV / resume site for **Roger Han** — Lead Mobile Engineer (native iOS, React Native, Flutter, AI-assisted delivery).

Live: https://roger--han.github.io

Static single-page site built on the [Start Bootstrap – Resume](https://startbootstrap.com/theme/resume) template (MIT). No build step — GitHub Pages serves the files directly from `main`.

## Edit

- Content: `index.html`
- Styles: `css/styles.css` (theme) + the `<style>` block in `index.html` (overrides)
- CV PDF: `assets/pdf/Roger_Han_CV_2026.pdf` (linked from the Download button)

Push to `main` and the live site updates within ~1 minute.
